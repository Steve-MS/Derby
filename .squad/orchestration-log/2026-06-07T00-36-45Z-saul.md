## 3 Hard Publish Blockers Identified

### 1. T-1hr Gate Timing (Derby Check Fired 39 Minutes Late)

**Finding:** Hourly watchdog ticks are insufficient granularity. Pre-race check fired at T−21min instead of T−60min.

**Root cause:** Hourly cron/scheduler pattern; no explicit named event at card-lock (18:00 BST).

**Mitigation:** Explicit named events at card-lock time instead of hourly ticks. Requires:
- Card lock gate at exact timestamp (not "next hourly check")

**Risk:** Race-day decisions truncated, offline analysis may miss fresh market moves or last-minute NR declarations.

### 2. Silent Completion Defence — Livingston-3 Output Sat 7+ Hours Unread

**Finding:** Livingston's baseline-gate output completed successfully at 07:08 BST but was not monitored/consumed until 14:00+ BST (7+ hours silent).

**Root cause:** Platform has known ~7–10% silent-completion success bug; spawn acknowledgement pattern not implemented. Output sits on filesystem unread.

**Mitigation:** Dual defence required:
- Filesystem polling: Check output file timestamp + size hourly; alert if >1hr stale
- Spawn ack pattern: Scribe or listener should acknowledge consumption within T+5min; escalate if no ack received

**Risk:** Stale-runner failures, NR cascades, market-move signal misalignments not detected until manual review.

---

### 3. HTML Header Staleness — Recurring Manual Patch Class

**Finding:** Report HTML header (`portfolio_summary`) not synced with JSON `portfolio_summary` after scoring injections. Manual CSS patch class required after every score update.

**Root cause:** Header computed at template render-time (not from JSON source); if JSON is updated and template re-rendered, header has stale values.

**Mitigation:** Compute header from JSON at render-time instead of fixed template. Requires:
- Move `portfolio_summary` computation into Jinja2 template filter
- Recompute header every time `src/report.py` renders HTML (no stale values)
- Remove manual CSS patch class

**Risk:** Misleading portfolio summary in race-day reports; team confusion about current bankroll state.
